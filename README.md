# ThinkBoard
A web application for managing notes with features like editing, deleting, and cloud storage.
ThinkBoard is a web application that allows users to create, edit, and manage notes. 
Notes are stored in a MongoDB database, and the app uses React for the frontend and Node.js for the backend.

## Features
- Create new notes
- Edit existing notes
- Delete notes
- Error handling and rate limiting
- Secure data storage in MongoDB

## Technologies
- Frontend: React, TailwindCSS, Vite
- Backend: Node.js, Express.js
- Database: MongoDB
- Others: Upstash Redis, Axios
